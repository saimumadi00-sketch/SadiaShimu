# — Academic Portfolio

A static academic portfolio website built with HTML, CSS, and vanilla JavaScript.

The site is designed to work on GitHub Pages or any static hosting provider. There is no build step required for the public portfolio.

## Files

```text
portfolio/
├── index.html
├── css/
│   └── main.css
├── js/
│   └── main.js
├── images/
│   ├── portrait.jpg
│   ├── gallery-1.jpg
│   ├── gallery-2.jpg
│   ├── gallery-3.jpg
│   ├── gallery-4.jpg
│   ├── gallery-5.jpg
│   └── gallery-6.jpg
└── README.md
```

## Images

Add the real photos later using these exact paths:

- `images/portrait.jpg`
- `images/gallery-1.jpg`
- `images/gallery-2.jpg`
- `images/gallery-3.jpg`
- `images/gallery-4.jpg`
- `images/gallery-5.jpg`
- `images/gallery-6.jpg`

If any of these files are missing, the site automatically shows clean placeholder images instead of broken image icons.

## Running Locally

Open `index.html` directly in a browser.

The map uses Leaflet and OpenStreetMap tiles from a CDN, so it loads when an internet connection is available.

## Deploying

For GitHub Pages, upload the static files and folders to a repository, then enable Pages in the repository settings.

No npm install, backend server, admin panel, or CMS is required for the static portfolio.
